docker常用命令

进入容器内部使用控制台

docker exec -it 容器id /bin/bash

查看当前容器

  1. 查看正在运行的容器 docker container ls
  2. 查看所有容器 `docker container ls -a ``

停止容器

  1. 指定 CONTAINER ID 停止容器 docker stop <CONTAINER ID>
  2. 指定容器名称停止容器 docker stop <CONTAINER NAME>

启动已经停止的容器

  1. 指定容器 CONTAINER ID 启动容器 docker start <CONTAINER ID>
  2. 指定容器名称启动容器 docker start <CONTAINER NAME>

重启已经运行的容器

  1. 指定容器 CONTAINER ID 重启容器 docker restart <CONTAINER ID>
  2. 指定容器名称重启容器 docker restart <CONTAINER NAME>

删除本地镜像

docker rmi [OPTIONS] IMAGE [IMAGE...] OPTIONS说明：

-f :强制删除；

–no-prune :不移除该镜像的过程镜像，默认移除；
